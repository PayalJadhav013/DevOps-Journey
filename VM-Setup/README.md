# Virtual Machine (VM) Setup

## Objective

Learn virtualization concepts and automate virtual machine provisioning using Vagrant.

---

## Topics Covered

- Introduction to Virtualization
- Hypervisors
- Oracle VM VirtualBox
- Manual VM Creation
- Vagrant
- Vagrantfile
- VM Provisioning
- SSH into VM

---

## Tools Used

- Oracle VM VirtualBox
- Vagrant
- Ubuntu

---

## Hands-on Activities

- Installed Oracle VM VirtualBox
- Created Ubuntu Virtual Machine manually
- Installed Vagrant
- Created VM using Vagrant
- Connected using SSH
- Managed VM lifecycle

---

## Learning Outcome

- Understood virtualization concepts.
- Learned manual VM creation.
- Automated VM provisioning using Vagrant.
- Practiced VM lifecycle management.
